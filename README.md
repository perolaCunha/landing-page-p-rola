Projeto Movimentando

O "Projeto Movimentando" visa ajudar pessoas que não possuem carros, motoss, ou condições para acessar algum transporte, ou até mesmo estudantes. Seria promovido por meio de doações, participação do governo e parcerias com empresas de transporte. 

A ODS relacionada é a de número 10( Redução das Desigualdades).
